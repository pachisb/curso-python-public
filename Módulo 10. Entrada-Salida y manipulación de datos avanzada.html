      ## UTF-8
      Es la codif. de caracteres m√°s habitual en Web y Linux

      # Avanzado: bytes()

      # Comprensiones (comprehensions)
      ## Expresiones generadoras
      Avanzado: Generator expresions
      Son una notaci√≥n abreviada para escribir un generador:
      ( expression for item in list if conditional )
      .
      Pueden sustituir a cosas como map(), lambdas, etc. de forma m√°s sencilla
      Al ser un generador, se van produciendo los valores "a demanda"...
      No son un bucle sino una expresi√≥n, aunque "se parecen" y no importa mucho
      La parte del if es opcional. M√°s detalles y doc. de referencia:
      : https://docs.python.org/3/reference/expressions.html#generator-expressions

      Ejemplos:

        mayusc = (str.upper(x) for x in mezclados if len(x) > 3)
        print(list(mayusc))
      .
        primeros_cuadrados = ( i**2 for i in range(10) )
        for i in primeros_cuadrados:
            print(i)
      .
        for i in ( i**2 for i in range(10) ): # lo mismo!
            print(i)
      .
      NOTA: si no es ambiguo, a veces se pueden omitir los par√©ntesis

      ## Comprensiones de listas
