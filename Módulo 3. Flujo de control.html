      # Formateado (estilo) del c√≥digo
      Estilo recomendado: PEP 8. Resumen:
      - Indentaci√≥n con 4 espacios, no tabuladores
      - L√≠neas de menos de 80 caracteres
      - Espacios tras las comas y entre operadores
      - No espacios tras/pre par√©ntesis (funciones, listas...)
      - Comentarios preferiblemente en una l√≠nea separada
      - Separar funciones, clases, y bloques grandes con l√≠neas en blanco
      - Nombres consistentes: variables_y_funciones, EjemploClases, CONSTANTES
      Referencia: The Python Tutorial (docs.python.org)
      M√°s detalles: PEP 8

      ## break en bucles for y while
      Se utiliza para interrumpir el bucle m√°s "cercano"
        total = 0
        while True:
            n = input("N√∫mero siguiente (en blanco para acabar)? ")
            if n.strip() == "":  # nada o s√≥lo espacios, \n, etc.
                break
            total += int(n)
        # despu√©s del break, se contin√∫a aqu√≠
        print("La suma total es: ", total)
      .
      Nota: en otros lenguajes adem√°s hay m√°s tipos de bucles (ej. los que
      comprueban la condic. al final), pero en Python se debe hacer siempre
      lo equivalente a base de usar break

      ## continue en bucles for y while
      Para continuar con la siguiente iteraci√≥n del bucle m√°s "cercano"
      Ejemplo:

            if num % 2 == 0:
                print("Found an even number", num)
                continue
            print("Found an odd number", num)

      ## else en bucles for y while
      Se ejecuta al final, s√≥lo si no se interrumpi√≥ con break
      Ejemplo:

      .
      No confundir con el else de las sentencias if...
