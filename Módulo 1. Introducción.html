      # Conceptos b√°sicos de
      # computaci√≥n y programaci√≥n
      Algoritmo = descripci√≥n de la soluci√≥n de un problema

      Codificaci√≥n = adaptar el algoritmo a un lenguaje, dando
      como resultado un programa
      Programa = conjunto de instrucciones comprensibles por
      un computador que permite ordenar la secuencia de pasos
      que resuelve un problema dado (usando informaci√≥n). Se
      le suele llamar c√≥digo fuente (source, o source code)

      Computador = m√°quina programable capaz de almacenar y
      ejecutar programas. Memoria + CPU + perif√©ricos (I/O)
      S. O. = conjunto de programas para facilitar el uso (ficheros,

      la programaci√≥n con uno o varios lenguajes (edici√≥n,
      depuraci√≥n, compilaci√≥n, ejecuci√≥n, control de versiones...)

      Dato = abstracci√≥n de la realidad, con un valor asociado, y
      que permite operar; con una representaci√≥n interna determinada
      Tipo de dato = dominio de posibles valores + representaci√≥n
