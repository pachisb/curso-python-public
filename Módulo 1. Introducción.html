      # Conceptos básicos de
      # computación y programación
      Algoritmo = descripción de la solución de un problema
      (incluyendo su información asociada), no ambigua, en
      una secuencia ordenada y finita de pasos
      Codificación = adaptar el algoritmo a un lenguaje, dando
      como resultado un programa
      Programa = conjunto de instrucciones comprensibles por
      un computador que permite ordenar la secuencia de pasos
      que resuelve un problema dado (usando información). Se
      le suele llamar código fuente (source, o source code)

      Computador = máquina programable capaz de almacenar y
      ejecutar programas. Memoria + CPU + periféricos (I/O)
      S. O. = conjunto de programas para facilitar el uso (ficheros,
      seguridad, herramientas) y controlar el funcionamiento
      de la máquina (HW: memoria, perif., asignación de recursos,
      scheduling...)
      IDE = Entorno de desarrollo integrado: programa que facilita
      la programación con uno o varios lenguajes (edición,
      depuración, compilación, ejecución, control de versiones...)

      Dato = abstracción de la realidad, con un valor asociado, y
      que permite operar; con una representación interna determinada
      Tipo de dato = dominio de posibles valores + representación
      + operadores asociados. Suele haber unos predefinidos siempre
      Ej. números naturales vs. enteros (o vs. reales o complejos)

      # Intérpretes y compiladores
      Ejecución de un programa:
      - Compilada: traducción previa a lenguaje máquina,
 generando un programa ejecutable o aplicación
      - Interpretada: paso a paso (cada instrucción),
 usando un intérprete necesariamente (entorno de
 ejecución). Se suele hablar de scripts (...)
      .
      Suele haber opciones "híbridas" (...)

      # Herramientas alternativas
      ## IPython
      Es un shell interactivo de Python mejorado
      : https://en.wikipedia.org/wiki/IPython
      : http://ipython.org/

      Cómo lanzarlo (varias opciones)
      Windows (recordar Windows+R como atajo para "Ejecutar"):
        ipython3.exe
        py.exe -m IPython
      Nota: la parte ".exe" se puede omitir. Mayúsculas en "IPython" importan!
      Nota: el primero puede que no funcione "tal cual" sin localizar su ruta.
      Normalmente estará en: %APPDATA%\Python\Python38\Scripts
      o en C:\Program Files... Lo mejor: crearnos accesos directos
      .
      Linux Ubuntu o Debian:
        ipython3
        python3 -m IPython
        python -m IPython  # sólo para Python 2.x!

      Cómo instalarlo de forma rápida para pruebas y desarrollo
      Windows / a veces Linux (depende del tipo de entorno):
        >>> import sys; import subprocess
        >>> subprocess.run([sys.executable, '-m',
              'pip', 'install', '--user', 'IPython'])
      .
      Linux (ojo, lo instala para todo el sistema y usuarios):
        sudo apt-get update
        sudo apt-get install python3-pip
        sudo pip3 install IPython
